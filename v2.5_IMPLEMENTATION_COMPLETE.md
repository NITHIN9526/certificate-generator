## ğŸ¯ How Users Use Categories

### Single Certificate
```
1. Select category (e.g., Sports)
2. System suggests appropriate templates
3. User chooses template
4. Fill certificate details
5. Download PNG or PDF
```

### Batch Processing - Same Category
```
1. Prepare Excel (Name, Event, Date, Issuer columns)
2. Leave Category column empty
3. Select category from form (e.g., Arts)
4. Upload Excel
5. All certificates use Arts category templates
```

### Batch Processing - Mixed Categories
```
1. Prepare Excel with Category column
2. Fill different categories per row
3. Upload Excel
4. System automatically selects best template for each
5. Each participant gets their category template
```

## Creating Batch Certificates by Category

### Excel Format for Categorized Batch Processing

**Columns Required:**
- Column A: Participant Name
- Column B: Event/Course Title
- Column C: Date
- Column D: Issuer/Organization
- Column E: Category (Optional - uses selected category if empty)

### Example Excel Data

```
Name              | Event                      | Date       | Issuer           | Category
John Smith        | 100m Sprint Final          | 2026-01-20 | City Sports Assoc | sports
Maria Garcia      | Landscape Painting Prize   | 2026-01-20 | Arts Guild        | arts
Ahmed Hassan      | Mathematics Olympiad Gold  | 2026-01-20 | Education Board   | academics
Lisa Chen         | Project Innovation Award   | 2026-01-20 | Tech Corporation  | innovation
David Brown       | Annual Leadership Award    | 2026-01-20 | Management Board  | leadership
```

### Processing Mixed Categories

The batch processor will automatically:
1. Read each participant's category from Column E
2. Select the appropriate template for that category
3. Generate certificates with category-specific designs
4. Create PNG and PDF for each participant

**Result:** Each participant gets a certificate matching their category!

## Tips for Best Results

### Choosing the Right Category

1. **Be Specific:** Choose the category that best matches the achievement
2. **Consider Culture:** Some categories have cultural significance
3. **Match Audience:** Consider what the certificate recipients expect
4. **Think Visually:** Choose based on the visual style you prefer

### Using Templates Effectively

1. **Professional Events:** Use Formal/Academic templates
2. **Creative Events:** Use Arts templates for more color
3. **Sports Events:** Use Sports templates for dynamic feel
4. **Mixed Audience:** Use Professional templates (Classic/Modern)
5. **Premium Events:** Use Gold/Excellence/Leadership templates

### Batch Processing with Categories

1. **Consistent Category:** Use same category for all participants
   - Faster to set up
   - All certificates match

2. **Mixed Categories:** Different category for each participant
   - More personalized
   - Requires Excel column with category
   - Each gets category-specific template

## Excel Tips for Categories

### Quick Setup

**Option 1: Single Category for All**
- Leave Column E empty
- System uses selected category
- Quick and simple

**Option 2: Different Categories**
- Fill Column E with category for each participant
- Choose any template (first suggested template per category used)
- More personalized

### Category Keywords

Use these exact keywords in Excel:
- `sports` - for Sports category
- `arts` - for Arts category
- `academics` - for Academics category
- `participation` - for Participation
- `excellence` - for Excellence
- `achievement` - for Achievement
- `leadership` - for Leadership
- `innovation` - for Innovation

### Invalid Categories

If a category name is misspelled or invalid:
- System will use the selected template default
- No error occurs
- Certificate still generates

## FAQ: Categories & Templates

**Q: Can I use any template with any category?**  
A: Yes! The category automatically suggests templates, but you can choose any template you prefer.

**Q: What if I don't specify a category in batch?**  
A: The selected category from the form will be used for all participants.

**Q: Do I need to use categories for batch processing?**  
A: No, categories are optional. You can use your Excel file without Column E.

**Q: Can I change category mid-batch?**  
A: Yes, add different categories in Column E for each participant, and each gets their own category's template.

**Q: Which template is best?**  
A: Choose based on:
- Event type (see decision tree above)
- Visual preference
- Audience expectations
- Organization branding

**Q: Can I create custom templates?**  
A: The current version has 15 built-in templates. Custom templates require CSS editing.

**Q: Why are some templates better for certain categories?**  
A: The templates use colors, fonts, and designs that match category themes and audience expectations.

**Q: Can I use Professional templates with any category?**  
A: Yes! Professional templates (Classic, Modern, Gold) work with any category and are universally recognized.

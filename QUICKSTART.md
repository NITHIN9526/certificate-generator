### Opening the Application

1. Open `index.html` in your web browser

## üìù Creating a Single Certificate

### Step 1: Enter Participant Details
- **Participant Name**: Enter the full name of the person receiving the certificate
- **Event/Course Title**: Enter what the certificate is for
- **Date**: Enter the date of completion
- **Organizer/Issuer**: Enter your organization or institution name

### Step 2: Add Visual Elements (Optional)
- **Logo**: Click "Choose File" to upload your organization logo
- **Signature**: Click "Choose File" to upload an authorized signature

### Step 3: Choose Template
- Select one of three professional templates:
  - **Classic**: Elegant and traditional
  - **Modern**: Contemporary and clean
  - **Gold**: Premium and formal

### Step 4: Preview & Export
- Click **Preview** button to see how it looks
- Download as **PNG** for web/email
- Download as **PDF** for printing

## üìä Creating Batch Certificates (Multiple)

### Step 1: Prepare Excel File
Create an Excel file (.xlsx) with this format:

| Name | Event | Date | Issuer |
|------|-------|------|--------|
| John Doe | Web Design | 2026-01-20 | Tech Academy |
| Jane Smith | Web Design | 2026-01-20 | Tech Academy |

**Or use this sample data structure:**
```
Column A: Full names of all participants
Column B: Event/Course title (can be same for all)
Column C: Completion date (can be same for all)
Column D: Issuer/Organization (can be same for all)
```

### Step 2: Upload Excel File
1. Scroll down to "Batch Certificate" section
2. Click on "Upload Excel File"
3. Select your prepared Excel file (.xlsx)

### Step 3: Process Certificates
1. Click **"Process & Download All"** button
2. Wait for the progress bar to complete
3. A ZIP file will download automatically

### Step 4: Extract & Use
1. Extract the ZIP file
2. You'll find PNG and PDF versions for each participant
3. Print, email, or share the certificates

## ‚ùì Frequently Asked Questions

**Q: Can I modify the certificate design?**  
A: The current templates are pre-designed. You can choose between three templates and customize the text content.

**Q: What file formats are supported?**  
A: Excel files (.xlsx) for batch processing. PNG and PDF for downloads.

**Q: Can I use the certificates for commercial purposes?**  
A: Yes! These certificates are yours to use as needed.

**Q: How many participants can I process at once?**  
A: Theoretically unlimited, but 100-200 per batch is recommended for best performance.

**Q: What if I make a mistake in the Excel file?**  
A: The app will show an error message. Fix the file and try again.

**Q: Can I download certificates in a different format?**  
A: Currently PNG and PDF are supported. SVG export may be added in future versions.

## üÜò Troubleshooting

**Buttons not working?**
- Refresh the page (Ctrl+R or Cmd+R)
- Clear browser cache
- Try a different browser

**Can't upload Excel file?**
- Ensure file is .xlsx format
- Try opening and re-saving it in Excel
- File might be corrupted - recreate it

**Missing progress bar?**
- Ensure JavaScript is enabled
- Try a modern browser (Chrome, Firefox, Safari, Edge)

**Certificates look wrong?**
- Check your input data for special characters
- Ensure logo/signature files are valid images
- Try a different template
